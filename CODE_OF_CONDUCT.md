# Code of Conduct

This project is committed to providing a welcoming and harassment-free experience for everyone.

## Be Respectful

- Treat others with respect and empathy.
- Assume good intent; ask clarifying questions.
- Focus on constructive feedback.

## Unacceptable Behavior

- Harassment, discrimination, or hateful language
- Personal attacks or sustained disruption
- Publishing private information without consent

## Reporting

If you experience or witness unacceptable behavior, please contact the maintainers via the project issues or email listed in the repository profile.

## Enforcement

Project maintainers may remove or edit content, temporarily or permanently ban contributors, or take other appropriate action to ensure a safe environment.

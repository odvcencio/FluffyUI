# API Reference

FluffyUI is documented via GoDoc and package-level guides in `docs/api/`.

## Quick access

- `docs/api/widgets.md`
- `docs/api/graphics.md`
- `docs/api/animation.md`
- `docs/api/agent.md`

## GoDoc

Run `go doc` locally or visit `pkg.go.dev` once the module is published.

Example:

```bash
go doc github.com/odvcencio/fluffyui/widgets
```
